# Belajar HTML Dasar

Project ini merupakan latihan membuat website sederhana menggunakan HTML dan CSS dasar.

## 📌 Deskripsi

Website ini berisi:
- Header
- Navigation Bar
- Section konten (Tentang Kami & Layanan)
- Footer
- Styling menggunakan CSS internal

Project ini cocok untuk pemula yang ingin belajar struktur dasar HTML.

## 🛠️ Teknologi yang Digunakan

- HTML5
- CSS3

## 📂 Struktur Project

```

belajar-html/
│
├── index.html
└── README.md

````

## 🚀 Cara Menjalankan

1. Clone repository ini:
   ```bash
   git clone https://github.com/pito125/belajar-html.git
````

2. Masuk ke folder project:

   ```bash
   cd belajar-html
   ```

3. Buka file `index.html` di browser.

## 🎯 Tujuan Pembelajaran

* Memahami struktur dasar HTML
* Mengenal tag semantic seperti `<header>`, `<nav>`, `<section>`, `<footer>`
* Menggunakan CSS sederhana untuk styling

Dibuat sebagai latihan belajar HTML.
